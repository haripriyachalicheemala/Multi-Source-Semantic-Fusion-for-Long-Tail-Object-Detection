# Multi-Source Semantic Fusion for Long-Tail Object Detection

 Overview

This repository contains the implementation, thesis documentation, figures, experimental tables, and evaluation pipeline for the research thesis:

“Multi-Source Semantic Fusion for Long-Tail Object Detection”

The research focuses on improving long-tail and generalized zero-shot object detection using semantic information fusion strategies and CLIP-based semantic representations.

---

# Research Contributions

- Multi-source semantic fusion
- Long-tail object detection
- Generalized zero-shot detection (GZSD)
- Semantic prompt fusion
- Adaptive fusion strategies
- Prior-aware fusion
- Uncertainty estimation
- Robustness and stability analysis
