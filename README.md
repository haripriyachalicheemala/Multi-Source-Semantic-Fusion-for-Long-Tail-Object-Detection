\# Prior-Aware Multi-Source Semantic Evidence Fusion for Long-Tail Open-Vocabulary Object Detection



\## Overview



This repository contains the final code, figures, tables, results, proposal, and manuscript for the project:



\*\*Prior-Aware Multi-Source Semantic Evidence Fusion for Long-Tail Open-Vocabulary Object Detection\*\*



The study investigates semantic evidence fusion for long-tail open-vocabulary object detection using OWL-ViT on LVIS and COCO-based evaluation settings. The work focuses on improving detection behavior for rare and long-tail categories through multi-source semantic prompts, fusion strategies, uncertainty analysis, prompt complementarity, and prior-aware evidence weighting.

\## Research Questions



The project is organized around six research questions:



1\. \*\*RQ1:\*\* Does multi-source semantic evidence improve open-vocabulary detection compared with primary prompts?

2\. \*\*RQ2:\*\* Which semantic fusion strategy performs best across datasets and long-tail groups?

3\. \*\*RQ3:\*\* How do uncertainty and semantic conflict relate to false-positive risk?

4\. \*\*RQ4:\*\* Which semantic sources contribute complementary evidence?

5\. \*\*RQ5:\*\* Does prior-aware semantic fusion improve long-tail detection behavior?

6\. \*\*RQ6:\*\* How does the complete framework evolve from baseline prompting to prior-aware fusion?



\## Methodology Summary



The framework evaluates:



\* Primary prompt detection

\* Multi-source semantic evidence generation

\* Semantic evidence fusion

\* Fusion strategy comparison

\* Threshold sensitivity analysis

\* Uncertainty and conflict analysis

\* Prompt ablation and complementarity analysis

\* Prior-aware long-tail fusion

\* Dataset-level and frequency-group-level framework evolution
