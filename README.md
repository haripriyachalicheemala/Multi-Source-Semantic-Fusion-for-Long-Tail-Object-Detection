\# Multi-Source Semantic Fusion for Long-Tail Object Detection



\## Overview



This repository contains the implementation, thesis, experimental figures, and evaluation tables for research on:



\*\*Multi-Source Semantic Fusion for Long-Tail Object Detection\*\*



The project investigates semantic fusion strategies for improving long-tail and zero-shot object detection using CLIP-based representations.
