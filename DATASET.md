\# Dataset Information



This project uses publicly available datasets for long-tail and generalized zero-shot object detection experiments.



\---



\# Datasets Used



\## 1. LVIS v1.0



Official Website:

https://www.lvisdataset.org/



The LVIS dataset was used for long-tail object detection evaluation and semantic prior experiments.



\---



\## 2. COCO-ZSD



Official Website:

https://cocodataset.org/



Used for generalized zero-shot detection experiments and semantic fusion evaluation.



\---



\# Dataset Access



The datasets were accessed through Kaggle and online dataset integration during experimentation.



Due to GitHub storage limitations, full datasets are NOT included in this repository.
